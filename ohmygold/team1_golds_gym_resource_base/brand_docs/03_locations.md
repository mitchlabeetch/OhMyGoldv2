## 6. Club Access Protocol

### Digital Entry System

Gold's Gym France uses a **QR code access system** managed through the mobile app:

1. Download the Gold's Gym France app
2. Log in with member credentials
3. Generate unique QR code for each visit
4. Scan at club entrance turnstile
5. Code is non-transferable and unique per access

> *"Pour acceder a tous les clubs Gold's Gym, un controle d'acces par code QR sera utilise. Pour ce faire, le membre devra telecharger l'application Gold's Gym qui generera un code QR (unique et non transferable) different a chaque acces."*

### Access Requirements
- Valid, up-to-date membership
- App downloaded and QR code generated
- Photo ID may be required for verification
- No other entry method is permitted

## 11. Key Location Insights for App Design

### Critical UX Considerations

1. **24/7 Access at Thiais** -- The app must support QR code generation at any time of day
2. **Two-Club System** -- Members can access both clubs with any subscription; the app should show both locations
3. **Proximity to Disneyland (Val d'Europe)** -- May attract international tourists; consider multi-language support
4. **Trial Session Booking** -- App should support trial session reservations
5. **Digital-First Access** -- No physical cards; everything is QR-code based through the app
6. **Locker Management** -- Daily use only; app could include locker availability indicators
